Redefinindo as propriedades padrões dos navegadores

Reset CSS é uma folha de estilos que irá sobrepor todas as propriedades CSS que os navegadores colocam por padrão nos elementos, ajudando com que os sites tenham o mesmo design em diferentes navegadores.
