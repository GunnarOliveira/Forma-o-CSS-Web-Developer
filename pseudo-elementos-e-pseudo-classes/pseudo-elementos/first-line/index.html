Pseudo-elementos

Usados para adicionar elementos extras na página, sem a necessidade de declará-lo no HTML ou para também para selecionar uma parte de algum elemento.

Identificado por: ::palavra-chave

Pseudo-elemento - ::first-line

Seleciona a primeira linha de um texto

Propriedades disponíveis:

  • color
  • background
  • fonte
  • Propriedade de texto
