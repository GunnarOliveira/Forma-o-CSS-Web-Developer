Pseudo-elementos

Usados para adicionar elementos extras na página, sem a necessidade de declará-los no HTML ou para selecionar uma parte de algum elemento.

Identificado por ::palavra-chave

Pseudo-elemento - ::first-letter

Seleciona a primeira letra da primeira linha

propriedades disponíveis:

  • color
  • background
  • margin
  • padding
  • border
  • font
  • Propriedade de texto
